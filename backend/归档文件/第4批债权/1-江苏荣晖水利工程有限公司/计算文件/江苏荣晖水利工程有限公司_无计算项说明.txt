python /root/debt_review_skills/universal_debt_calculator_cli.py simple \

步骤二：第二阶段利息（2019年8月20日 - 2025年9月16日）
────────────────────────────────────────────
使用calculator工具的lpr模式（LPR浮动利率）：

python /root/debt_review_skills/universal_debt_calculator_cli.py lpr \
  --principal 4314677 \
  --start-date 2019-08-20 \
  --end-date 2025-09-16 \
  --multiplier 1.0 \
  --lpr-term 5y+

说明：
  - principal：工程款本金4,314,677元
  - start-date：LPR改革日2019年8月20日
  - end-date：破产受理日（停止计息日）2025年9月16日
  - multiplier：1.0倍LPR（无合同约定，按法定标准）
  - lpr-term：5y+（5年期以上LPR）

LPR期限选择理由：
  • 债务发生于2012年，履行期限较长（超过5年）
  • 建设工程价款债务通常具有长期性质
  • 5年期以上LPR更符合长期债务的利息水平
  • 相比1年期LPR更为保守（利率更低，符合"就低原则"精神）

步骤三：合计利息并应用就低原则
────────────────────────────────────────────
计算总利息 = 第一阶段利息 + 第二阶段利息

应用就低原则：
  如果 计算总利息 > 1,479,934.21元（债权人申报金额）
    则确认金额 = 1,479,934.21元
  如果 计算总利息 ≤ 1,479,934.21元
