# 债权分析报告模板与标准

## 📋 文档说明
本文档为债权分析员专用参考文件，包含：
- 输出格式标准
- 报告模板
- 完整示例
- 计算过程表格文件说明

# 第一部分：输出格式标准

## 1. 债权基础法律关系确认格式
```
独立债权基础法律关系确认：
- 关系数量：[X]个
- 具体清单：
  1. [关系1：合同名称/案号等标识]
  2. [关系2：合同名称/案号等标识]
- 法律关系性质：[合同类/法律文书类/其他]
```

## 2. 金额项目拆解清单格式

**表格格式要求：**
- 表格前后必须有空行
- 表格标题后必须空一行再开始表格

**金额项目清单：**

| 序号 | 项目描述 | 申报金额 | 法律性质 | 证据支持情况 | 计算需求 |
|------|----------|----------|----------|--------------|----------|
| 1 | [具体描述] | [金额] | 本金/利息/费用 | 有/不足/缺失 | 固定/需计算 |
| 2 | [具体描述] | [金额] | 本金/利息/费用 | 有/不足/缺失 | 固定/需计算 |

## 3. 履行期限判断格式

**履行期限判断：**

| 项目 | 履行期限 | 届满日 | 破产受理日 | 是否计息 |
|------|----------|--------|------------|----------|
| [项目名称] | [期限描述] | [日期] | 2025-05-08 | 是/否 |

## 4. 计算参数格式

### 4.1 固定金额类项目
```
（X）[项目名称]：[确定金额]元；
```

### 4.2 借贷类合同利息（固定利率）
```
（X）[项目名称]：以[计算基数]元为计算基数，自[起息日]起至[止息日]止，按[年利率]%的年利率计算（基准天数[360/365]天），计算可得共[]元；
```

### 4.3 借贷类合同利息（LPR浮动利率）
```
（X）[项目名称]：以[计算基数]元为计算基数，自[起息日]起至[止息日]止，按[LPR期限类型]LPR的[倍数]倍计算（基准天数[360/365]天），计算可得共[]元；
```

### 4.4 普通合同逾期利息
```
（X）[项目名称]：以[计算基数]元为计算基数，自[起息日]起至[止息日]止，按[年利率]%的年利率计算（基准天数[360/365]天），计算可得共[]元；
```

### 4.5 迟延履行期间债务利息
```
（X）迟延履行期间的加倍部分债务利息：以[计算基数]元为计算基数，自[起息日]（[履行期限届满日]次日）至[止息日]止（破产受理日前一日），以日利率万分之1.75为利率，计算可得共[]元；
```

## 上海金桥信息股份有限公司债权分析报告

# 上海金桥信息股份有限公司债权分析报告

## 一、债权基础法律关系确认

基于《上海金桥信息股份有限公司事实核查报告》，确认：
- 独立债权关系数量：1个
- 债权类型：法律文书确认类债权
- 具体识别：（2025）沪0120民初947号民事调解书确认的债权
- 法律关系性质：建设工程施工合同纠纷

## 二、金额项目拆解分析

### 2.1 金额项目清单
根据（2025）沪0120民初947号调解结果，被告同意支付原告工程款685,562.00元及案件受理费5,328.00元。

| 序号 | 项目描述 | 申报金额 | 法律性质 | 证据支持情况 | 计算需求 |
|------|----------|----------|----------|--------------|----------|
| 1 | 工程款 | 685,562.00元 | 本金 | 有（调解书确认） | 固定 |
| 2 | 案件受理费 | 5,328.00元 | 费用 | 有（调解书确认） | 固定 |
| 3 | 迟延履行期间加倍部分债务利息 | 5,843.98元 | 劣后债权 | 需计算验证 | 需计算 |

### 2.2 拆解说明
根据调解书内容，债权构成清晰：
- 工程款本金685,562.00元为主债权
- 案件受理费5,328.00元为诉讼费用
- 迟延履行期间的加倍部分债务利息需根据分期付款安排计算

## 三、履行期限判断

| 项目 | 履行期限 | 届满日 | 破产受理日 | 是否计息 |
|------|----------|--------|------------|----------|
| 第一期工程款(120,000元) | 2025年3月20日前 | 2025-03-20 | 2025-05-08 | 是 |
| 后续工程款(565,562元) | 2025年4月30日起分期 | 2025-04-30 | 2025-05-08 | 是 |
| 案件受理费(5,328元) | 2026年2月28日前 | 2026-02-28 | 2025-05-08 | 否 |

### 起息日确定
- 调解书作出日：2025年2月20日
- 调解书生效日：2025年2月20日（调解书当日生效）
- 第一期付款履行期：2025年3月20日
- 第一期起息日：2025年3月21日
- 后续付款履行期：2025年4月30日  
- 后续起息日：2025年5月1日

## 四、利息计算过程

### 4.1 计算命令记录

```bash
# 第一期工程款迟延履行利息
python universal_debt_calculator_cli.py delay --principal 120000 --start-date 2025-03-21 --end-date 2025-05-07

# 后续工程款迟延履行利息  
python universal_debt_calculator_cli.py delay --principal 565562 --start-date 2025-05-01 --end-date 2025-05-07

# 生成Excel格式的计算过程表
python universal_debt_calculator_cli.py delay --principal 120000 --start-date 2025-03-21 --end-date 2025-05-07 --excel-output "上海金桥信息_迟延履行利息计算.xlsx" --sheet-name "第一期工程款" --debtor "上海金桥信息股份有限公司"

python universal_debt_calculator_cli.py delay --principal 565562 --start-date 2025-05-01 --end-date 2025-05-07 --excel-output "上海金桥信息_迟延履行利息计算.xlsx" --sheet-name "后续工程款" --debtor "上海金桥信息股份有限公司" --append
```

### 4.2 计算结果汇总

| 计算基数 | 起息日 | 止息日 | 计算天数 | 计算结果 | 备注 |
|----------|--------|--------|----------|----------|------|
| 120,000.00元 | 2025-03-21 | 2025-05-07 | 48天 | 1,008.00元 | 第一期工程款 |
| 565,562.00元 | 2025-05-01 | 2025-05-07 | 7天 | 4,771.72元 | 后续工程款 |
| **合计** | | | | **5,779.72元** | |

**与申报金额对比：**
- 申报金额：5,843.98元
- 计算结果：5,779.72元  
- 按"就低原则"：确认5,779.72元

## 五、诉讼时效分析

### 5.1 时效判断
本债权基于生效法律文书（民事调解书），已经司法确认，不涉及诉讼时效问题。

## 六、执行时效分析

### 6.1 执行时效判断
- 执行依据：（2025）沪0120民初947号民事调解书
- 履行期限：第一期2025年3月20日前，后续分期至2026年2月28日
- 时效起算：2025年3月20日（第一期履行期限届满日）
- 中断事由：无
- 最终届满：2027年3月20日
- 分析结论：**未超过执行时效**

### 6.2 详细说理
根据民事调解书约定，第一期120,000元应于2025年3月20日前支付。根据法律规定，申请执行的期间为二年，从法律文书规定的履行期间的最后一日起计算。从第一期履行期限届满日2025年3月20日起算，执行时效届满日为2027年3月20日。债权人于破产受理后（2025年5月）申报债权，距离执行时效届满还有近两年时间，故未超过执行时效。

## 七、审查确认情况

### 7.1 申报与确认对比表

| 项目 | 申报金额 | 确认金额 | 性质 | 确认说明 |
|------|----------|----------|------|----------|
| 本金（工程款） | 685,562.00元 | 685,562.00元 | 普通债权 | 调解书确认，与申报一致 |
| 案件受理费 | 5,328.00元 | 5,328.00元 | 普通债权 | 调解书确认，与申报一致 |
| 迟延履行利息 | 5,843.98元 | 5,779.72元 | 劣后债权 | 经计算确认 |
| **合计** | **696,733.98元** | **696,669.72元** | | |

### 7.2 债权性质确认
- 优先债权：0元
- 普通债权：690,890.00元（本金685,562.00元 + 案件受理费5,328.00元）
- 劣后债权：5,779.72元（迟延履行期间的加倍部分债务利息）

## 八、审查结论

经审查，申报债权总额696,733.98元，确认债权总额696,669.72元。

### 确认明细：
1. **工程款本金**：申报685,562.00元，确认685,562.00元（普通债权）
2. **案件受理费**：申报5,328.00元，确认5,328.00元（普通债权）
3. **迟延履行利息**：申报5,843.98元，确认5,779.72元（劣后债权）

### 确认结果：
- 确认债权总额：696,669.72元
- 优先债权：0元
- 普通债权：690,890.00元
- 劣后债权：5,779.72元

### 特别说明：
关于案件受理费的迟延履行期间加倍部分利息：调解书明确"被告应于2026年2月28日前支付原告"案件受理费，支付期限截至破产受理日（2025年5月8日）尚未届满，故对这部分迟延履行期间加倍部分利息不予确认。
