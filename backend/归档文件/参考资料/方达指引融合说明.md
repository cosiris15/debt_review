### 参考文件: references/ (9个文件, 约3,700行)

#### 1. 主体与证据标准 (01, 573行)
- 债权主体资格审查
- 举证责任与证据层级
- **三大期间**: 诉讼时效、执行期间、保证期间
- 多期间协同审查方法

#### 2. 涉诉债权标准 (02, 518行)
- 法律文书层级认定
- 诉讼费用确认规则
- **迟延履行期间债务利息**: 一般部分 + 加倍部分
- 加倍部分劣后债权分类

#### 3. 利息计算标准 (03, 528行)
- 五大计算原则
- 分段计算方法（利率变化、部分清偿）
- 复利计算规则
- 利息与本金的分类规则

#### 4. 金融机构债权标准 (04, 456行)
- 金融机构资格认定
- 利率合理性标准（不适用民间借贷上限）
- 复利规则（针对利息，非本金）
- 罚息与复利不可叠加

#### 5. 民间借贷标准 (05, 483行)
- **砍头息禁止**（实际出借额为本金）
- **变相利息识别**（服务费、中介费）
- **利率上限**: 2020年前24% vs 2020年后LPR×4
- 综合成本上限（利息+罚息+费用总和）

#### 6. 担保类债权标准 (06, 456行)
- 担保物权完善要件
- **保证期间**: 6个月默认，除斥期间性质
- **越权担保**: 民法典担保解释第8条框架
- **让与担保**: 所有权转移形式担保
- **混合担保**: 民法典第392条顺序规则

#### 7. 特殊债权类型标准 (07, 300行)
- **融资租赁**: 虚构租赁物无效
- **保理**: 三种场景（供应商、债务人、担保人）+ 虚构应收账款严查
- **建设工程**: **18个月除斥期间** + 优先于抵押权

#### 8. 职工债权标准 (08, 180行)
- 超优先级（Article 113(1)第一顺位）
- 管理人主动调查（无需申报）
- **第三方垫付**: 一般第三方（第一顺位） vs 欠薪保障基金（第二顺位）

#### 9. 其他事项标准 (09, 464行)
- **个别清偿**: 6个月内可撤销（有例外）
- **无效清偿**: 受理后清偿自动无效
- **抵销权**: 互负债务 + 均到期 + 非禁止情形
- **取回权**: 所有权证明 + 物品存在
- **税费**: 税款本金（第二顺位） vs 滞纳金（普通） vs 罚款（劣后）

## 使用示例

### 触发场景1: 越权担保
```
用户提问: "这个公司提供的担保没有股东会决议，应该如何认定？"
→ 触发关键词: "担保" + "股东会决议"（暗示越权担保）
→ 自动加载: debt-review-legal-standards
→ 参考: 06_secured_claims_standards.md Part 5 (Ultra Vires Guarantee)
```

### 触发场景2: 建设工程优先权
```
用户提问: "这笔工程款是2020年到期的，现在2024年申报，还有优先权吗？"
→ 触发关键词: "工程款" + "优先权"
→ 自动加载: debt-review-legal-standards
→ 参考: 07_special_claims_standards.md Part 3 (18-month exclusion period)
→ 分析: 2020到期 + 18个月 = 2021年中到期 → 优先权已丧失
```

### 不触发场景: 常规买卖合同
```
用户提问: "这笔100万的货款，有合同、送货单、对账单，如何确认？"
→ 无复杂法律问题关键词
→ 不触发: debt-review-legal-standards
→ 使用: debt-claim-analysis常规流程即可
```
