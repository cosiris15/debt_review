# 债权预处理指南

## 概述

债权预处理是事实核查的第一步，目的是快速识别债权结构、评估复杂度、为后续核查提供导航。

**核心原则**：所有债权均需进行预处理，生成债权结构概览。

---

## 一、预处理工作流

### 1.1 标准流程

```
接收材料
    ↓
Step 1: 快速浏览材料（5-10分钟）
    ↓
Step 2: 识别债权类型和复杂度
    ↓
Step 3: 选择模板版本（简化版/完整版）
    ↓
Step 4: 填写债权结构概览
    ↓
Step 5: 生成法律关系图（如满足条件）
    ↓
移交详细事实核查
```

### 1.2 时间预算

| 债权类型 | 预处理时间 | 说明 |
|----------|------------|------|
| 简单债权 | 10-15分钟 | 单笔、无担保或简单担保 |
| 普通金融债权 | 20-30分钟 | 2-3笔、有担保 |
| 复杂金融债权 | 30-60分钟 | 多笔、复杂担保、债权转让 |
| 超长材料 | 60-90分钟 | >200页，需要详细索引 |

## 二、快速浏览技巧

### 2.1 材料阅读顺序

**第一优先级**（必读，确定基本框架）：
1. 债权申报书首页 → 债权人信息、申报总额
2. 债权申报书金额明细页 → 本金/利息/其他分项
3. 债权申报书债权性质勾选 → 担保/普通债权

**第二优先级**（识别债权结构）：
4. 主合同首页和签章页 → 合同编号、签订日期、合同金额
5. 担保合同首页 → 担保人、担保方式
6. 判决书/调解书首页和判项 → 案号、判决金额

**第三优先级**（完善细节）：
7. 抵押/质押登记文件
8. 债权转让协议（如有）
9. 其他履行证据

### 2.2 快速识别关键信息

**债权人类型识别**：
```
银行：名称含"银行"、"Bank"
信托：名称含"信托"、"Trust"
AMC：名称含"资产管理"、"东方/长城/华融/信达"
融资租赁：名称含"融资租赁"、"金融租赁"
保理：名称含"保理"、"商业保理"
```

**多笔债权识别**：
```
- 申报书有"合计"行
- 存在多个主合同编号
- 金额明细有多行本金
- 多个判决书/多个案号
```

**债权转让识别**：
```
- 申报书债权人与合同相对方不一致
- 存在《债权转让协议》
- 存在《债权转让通知书》
- 申报书备注提及"受让"
```

## 三、版本选择决策树

```
开始
  │
  ├─ 债权人是否为金融机构？
  │   ├─ 是 → 完整版
  │   └─ 否 ↓
  │
  ├─ 是否有多笔债权（≥2笔）？
  │   ├─ 是 → 完整版
  │   └─ 否 ↓
  │
  ├─ 是否有复杂担保（担保人≥2 或 担保物≥2 或 混合担保）？
  │   ├─ 是 → 完整版
  │   └─ 否 ↓
  │
  ├─ 是否存在债权转让？
  │   ├─ 是 → 完整版
  │   └─ 否 ↓
  │
  ├─ 材料是否超长（≥200页）？
  │   ├─ 是 → 完整版
  │   └─ 否 → 简化版
```

## 四、填写要点

### 4.1 简化版填写要点

**重点填写**：
- 债权人基本信息（完整照抄）
- 申报金额（分项和总额）
- 核心合同信息
- 担保情况勾选
- 材料索引

**可简化内容**：
- 不需要详细的担保矩阵
- 不需要债权转让链条
- 预处理结论简要即可

### 4.2 完整版填写要点

**必须详细填写**：
1. **各笔债权明细**：每笔都要单独列出
2. **担保关系矩阵**：明确每个担保对应哪些债权
3. **债权转让链条**：完整的转让历史
4. **材料索引**：精确到页码范围
5. **预处理结论**：核查重点和特殊注意事项

**关键判断**：
- 是否需要分批核查
- 复杂度评估
- 潜在法律风险点

## 五、金融债权特殊处理

### 5.1 银行债权

**常见结构**：
- 流动资金贷款（单笔或多笔）
- 授信额度下的循环贷款
- 银行承兑汇票垫款

**核查重点**：
- 授信合同 vs 具体借款合同的关系
- 利率约定（固定/浮动/LPR基点）
- 复利和罚息条款
- 多笔贷款的担保覆盖范围

### 5.2 信托债权

**常见结构**：
- 信托贷款
- 信托受益权转让
- 结构化融资

**核查重点**：
- 信托计划文件
- 委托人与受益人关系
- 担保增信安排

### 5.3 AMC（资产管理公司）债权

**常见结构**：
- 从银行受让的不良贷款
- 多轮转让的债权包

**核查重点**：
- 完整的转让链条
- 每次转让的通知情况
- 金融不良债权的特殊规则
- 原始债权的瑕疵是否继承

### 5.4 融资租赁债权

**常见结构**：
- 直接租赁
- 售后回租

**核查重点**：
- 租赁物所有权归属
- 租赁期限和租金计算
- 是否构成融资租赁还是借贷
- 租赁物的抵押/转让情况

### 5.5 保理债权

**常见结构**：
- 有追索权保理
- 无追索权保理
- 反向保理

**核查重点**：
- 基础交易的真实性
- 应收账款的转让通知
- 保理融资与应收账款的对应关系
- 回购/追索安排

## 六、超长材料处理

### 6.1 预处理阶段的分批策略

当材料超过200页时，预处理阶段需要：

1. **建立详细的材料索引**
   - 精确到每份文件的页码范围
   - 标注文件类型和对应的债权笔数

2. **识别核心材料与辅助材料**
   - 核心：合同、判决书、转让协议
   - 辅助：履行凭证、往来函件

3. **为后续分批核查做准备**
   - 建议分批方案
   - 各批次的内容边界

### 6.2 材料索引标准格式

```markdown
| 序号 | 材料名称 | 页码范围 | 类型 | 对应债权 | 重要性 |
|------|----------|----------|------|----------|--------|
| 1 | 债权申报书 | P1-P10 | 申报文件 | 全部 | 核心 |
| 2 | 借款合同（2020-001） | P11-P35 | 主合同 | 第1笔 | 核心 |
| 3 | 借款合同（2020-002） | P36-P60 | 主合同 | 第2笔 | 核心 |
| 4 | 保证合同 | P61-P80 | 担保合同 | 第1、2笔 | 核心 |
| 5 | 抵押合同 | P81-P95 | 担保合同 | 第1笔 | 核心 |
| 6 | 抵押登记证明 | P96-P100 | 登记文件 | 第1笔 | 核心 |
| 7 | 判决书（2023）X民初X号 | P101-P130 | 法律文书 | 第1笔 | 核心 |
| 8 | 银行流水 | P131-P200 | 履行证据 | 第1笔 | 辅助 |
| 9 | 催收函 | P201-P210 | 往来文件 | 全部 | 辅助 |
| ... | ... | ... | ... | ... | ... |
```

## 八、质量检查

### 8.1 预处理完成检查清单

- [ ] 债权结构概览已生成
- [ ] 版本选择正确（简化版/完整版）
- [ ] 债权人信息完整准确
- [ ] 申报金额分项与总额一致
- [ ] 各笔债权均已识别（如适用）
- [ ] 担保关系已梳理（如适用）
- [ ] 债权转让链条完整（如适用）
- [ ] 材料索引页码准确
- [ ] 预处理结论已填写
- [ ] 法律关系图已生成（如满足条件）

### 8.2 常见问题

**问题1：无法确定债权笔数**
- 解决：以主合同数量为准，一个主合同算一笔

**问题2：担保关系复杂难以梳理**
- 解决：先画出担保矩阵草图，再填入模板

**问题3：材料页码不连续**
- 解决：按实际页码标注，注明不连续情况

**问题4：债权转让链条不完整**
- 解决：标注缺失环节，在预处理结论中说明

## 九、与后续流程的衔接

### 9.1 移交详细事实核查

预处理完成后，移交给详细事实核查时应说明：

1. 已生成的文件列表
2. 债权复杂度和建议核查方式
3. 需要重点关注的问题
4. 材料缺失或不清晰之处

### 9.2 信息传递

债权结构概览中的以下信息将被后续流程使用：

- **事实核查**：材料索引、各笔债权结构
- **债权分析**：金额明细、担保矩阵
- **报告整理**：债权笔数、担保关系汇总
