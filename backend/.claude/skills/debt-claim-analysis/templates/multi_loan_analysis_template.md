# 多笔贷款债权分析报告模板

## 文档说明

本模板适用于同一债权人申报多笔贷款的债权分析报告。

**适用场景**：
- 银行/信托/AMC等金融机构债权
- 同一债权人≥2笔独立贷款
- 授信额度下多笔放款

**文件命名**：`{债权人名称}_债权分析报告.md`

**存放位置**：`工作底稿/`

## 填写规则

### 1. 占位符替换

- 所有 `[内容]` 格式的占位符必须替换为实际内容
- 金额统一使用"元"为单位，不使用"万元"
- 日期统一使用 YYYY-MM-DD 格式

### 2. 表格扩展

- 如债权笔数超过3笔，按相同格式继续添加
- 汇总表需对应调整行数

### 3. 计算文件

- 每笔债权的利息计算必须使用calculator工具
- 必须保存计算过程文件（Excel格式）
- 文件命名遵循标准

### 4. 就低原则

- 按笔独立适用
- 在确认金额栏注明依据
