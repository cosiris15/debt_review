# 金融多笔贷款债权分析指南

## 概述

本指南定义同一债权人申报多笔贷款时的分析流程和标准。适用于银行、信托、AMC、融资租赁等金融机构债权。

**核心原则**：
- 逐笔独立分析，分别计算
- 汇总确认，统一呈现
- 就低原则按笔独立适用
- 共同担保单独分析

---

## 一、适用场景

### 1.1 必须使用本指南的情况

| 场景 | 说明 |
|------|------|
| 同一债权人≥2笔贷款 | 如银行3笔流动资金贷款 |
| 授信额度下多笔放款 | 如授信5000万，分次放款 |
| 多个独立借款合同 | 各合同相互独立 |
| 债权转让包含多笔 | AMC受让的多笔不良贷款 |

### 1.2 不适用的情况

| 场景 | 处理方式 |
|------|----------|
| 单笔贷款 | 使用标准分析流程 |
| 展期/续贷（合同号不变） | 视为同一笔贷款处理 |
| 循环贷款（同一额度） | 按最终余额处理 |

---

## 二、分析流程

### 2.1 标准流程

```
Step 1: 读取债权结构概览
    ↓
Step 2: 确认债权笔数和各笔基本信息
    ↓
Step 3: 逐笔进行独立分析
    ├─ 本金分析
    ├─ 利息计算（使用calculator工具）
    └─ 担保分析
    ↓
Step 4: 汇总确认情况
    ↓
Step 5: 共同担保分析
    ↓
Step 6: 生成多笔分析报告
```

### 2.2 关键输入

从事实核查阶段获取：
- `{债权人名称}_债权结构概览.md` - 各笔债权基本信息
- `{债权人名称}_事实核查报告.md` - 详细事实信息
- `{债权人名称}_法律关系图.md` - 担保关系可视化（如有）

## 三、逐笔分析要点

### 3.1 本金分析

**每笔债权独立分析**：

| 分析项 | 说明 |
|--------|------|
| 合同本金 | 借款合同约定的借款金额 |
| 实际发放 | 银行流水证明的实际放款金额 |
| 已偿还 | 还款记录证明的已还本金 |
| 申报本金 | 债权人申报的本金金额 |
| 确认本金 | 取实际发放-已偿还与申报本金的较低者 |

**就低原则应用**：
```
确认本金 = min(证据证明本金, 申报本金)

示例：
证据证明：借款500万，已还100万，剩余400万
申报本金：380万
确认本金：380万（取较低者）
```

### 3.2 利息计算

**每笔贷款独立计算，必须使用calculator工具**

**利率确定优先级**：
1. 判决书/调解书确定的利率
2. 双方确认的利率（对账单、确认函）
3. 合同约定的利率
4. 无约定时适用LPR

**常见利息类型**：

| 利息类型 | 计算方式 | calculator模式 |
|----------|----------|----------------|
| 合同期内利息 | 按合同约定利率 | simple / lpr |
| 逾期利息 | 合同约定或LPR+30%-50% | simple / lpr |
| 罚息 | 在合同利率基础上加收 | penalty |
| 复利 | 利息计入本金计息 | compound |
| 迟延履行利息 | 判决生效后按规定计算 | delay |

**calculator工具调用示例**：

```bash
# 第1笔贷款利息计算
python universal_debt_calculator_cli.py lpr \
    --principal 5000000 \
    --start-date 2023-01-01 \
    --end-date 2024-05-19 \
    --multiplier 1.0 \
    --lpr-term 1y

# 第2笔贷款利息计算
python universal_debt_calculator_cli.py lpr \
    --principal 3000000 \
    --start-date 2023-06-01 \
    --end-date 2024-05-19 \
    --multiplier 1.3 \
    --lpr-term 1y
```

**利率期限选择**（LPR浮动）：

| 借款期限 | LPR期限选择 |
|----------|-------------|
| ≤1年 | 1y |
| >1年 | 5y |

### 3.3 担保分析

**每笔贷款的担保情况需单独说明**：

| 笔号 | 担保人/物 | 担保方式 | 担保范围 | 担保效力 |
|------|-----------|----------|----------|----------|
| 1 | 张三 | 连带保证 | 全部债务 | 有效 |
| 1 | XX厂房 | 抵押担保 | 全部债务 | 已登记 |
| 2 | 张三 | 连带保证 | 全部债务 | 有效 |
| 3 | XX股权 | 质押担保 | 本笔债务 | 已登记 |

## 四、汇总确认

### 4.1 汇总表格式

```markdown
## 汇总确认情况

| 笔号 | 合同编号 | 申报本金 | 确认本金 | 申报利息 | 确认利息 | 债权性质 | 备注 |
|------|----------|----------|----------|----------|----------|----------|------|
| 1 | 借字2020-001 | X元 | X元 | X元 | X元 | 担保债权 | |
| 2 | 借字2021-002 | X元 | X元 | X元 | X元 | 担保债权 | |
| 3 | 借字2022-003 | X元 | X元 | X元 | X元 | 普通债权 | 无担保 |
| **合计** | | **X元** | **X元** | **X元** | **X元** | | |
```

### 4.2 汇总计算规则

**金额汇总**：
```
汇总申报本金 = Σ(各笔申报本金)
汇总确认本金 = Σ(各笔确认本金)
汇总申报利息 = Σ(各笔申报利息)
汇总确认利息 = Σ(各笔确认利息)
```

**验证规则**：
- 汇总申报金额应与债权申报书总额一致
- 如不一致，需标注差异原因

### 4.3 债权性质汇总

```markdown
## 债权性质汇总

| 债权性质 | 笔数 | 本金合计 | 利息合计 |
|----------|------|----------|----------|
| 担保债权 | X笔 | X元 | X元 |
| 普通债权 | X笔 | X元 | X元 |
| **总计** | **X笔** | **X元** | **X元** |
```

## 五、共同担保分析

### 5.1 分析要点

当多笔贷款存在共同担保时，需要额外分析：

1. **担保覆盖范围**：
   - 哪些担保覆盖全部债务
   - 哪些担保仅覆盖特定笔债务

2. **担保物权顺位**：
   - 是否存在多个抵押权
   - 各抵押权的顺位关系

3. **保证责任分配**：
   - 多个保证人之间的责任分担
   - 物保与人保并存时的责任关系

### 5.2 共同担保说明格式

```markdown
## 共同担保情况分析

### 5.1 担保覆盖关系

| 担保 | 覆盖笔1 | 覆盖笔2 | 覆盖笔3 | 担保限额 |
|------|---------|---------|---------|----------|
| 保证人张三 | ✓ | ✓ | ✓ | 无限额 |
| 保证人李四 | ✓ | ✓ | | 500万元 |
| 抵押物A | ✓ | | | 评估价300万 |

### 5.2 担保分析结论

1. **保证人张三**为全部债务提供连带保证，无限额限制
2. **保证人李四**仅为第1、2笔债务提供保证，担保限额500万元
3. **抵押物A**仅担保第1笔债务，债权人对该笔债务享有优先受偿权
4. 第3笔债务无担保，为普通债权
```

## 六、特殊情况处理

### 6.1 部分笔债权已清偿

```markdown
## 第2笔债权分析

**合同信息**：借字2021-002，借款300万元

**清偿情况**：
- 根据还款记录，该笔债务已于2023年6月15日全部清偿
- 债权人在申报书中未申报该笔债权

**分析结论**：
- 该笔债权已清偿完毕，无需确认
- 汇总表中标注"已清偿"
```

### 6.2 部分笔债权有生效判决

```markdown
## 第1笔债权分析

**合同信息**：借字2020-001，借款500万元

**判决情况**：
- （2023）X民初X号判决书已生效
- 判决确认本金500万元，利息按LPR+50%计算至实际清偿日
- 判决同时确认担保人张三承担连带保证责任

**分析结论**：
- 本金：按判决确认的500万元
- 利息：按判决标准计算至破产受理日前一日
- 债权性质：担保债权（基于判决确认的担保效力）
```

### 6.3 部分笔债权诉讼时效存疑

```markdown
## 第3笔债权分析

**合同信息**：借字2018-003，借款200万元

**时效分析**：
- 借款到期日：2019年6月30日
- 最后一次催收记录：2020年3月15日
- 起诉日期：2023年8月1日
- 破产受理日：2024年5月20日

**分析结论**：
- 诉讼时效可能已届满
- 建议确认意见：【暂缓确认】
- 原因：距最后催收超过3年，未见时效中断证据
```

## 八、质量检查

### 8.1 检查清单

- [ ] 各笔债权均已独立分析
- [ ] 每笔利息计算均使用calculator工具
- [ ] 计算文件已保存（Excel格式）
- [ ] 汇总表金额与各笔合计一致
- [ ] 汇总申报金额与申报书一致
- [ ] 就低原则按笔独立适用
- [ ] 共同担保覆盖关系已分析
- [ ] 债权性质按笔正确分类

### 8.2 常见错误

| 错误类型 | 说明 | 正确做法 |
|----------|------|----------|
| 合并计算利息 | 将多笔本金合并计算 | 逐笔独立计算 |
| 就低原则整体适用 | 按总额适用就低原则 | 按笔独立适用 |
| 遗漏担保分析 | 未分析担保覆盖范围 | 编制担保覆盖矩阵 |
| 汇总金额错误 | 各笔合计与汇总不一致 | 交叉验证 |

## 九、与上下游流程的衔接

### 9.1 接收事实核查报告后

1. 首先阅读债权结构概览，确认笔数
2. 阅读法律关系图，理解担保关系
3. 阅读事实核查报告，获取详细事实

### 9.2 移交报告整理前

1. 确保各笔分析完整
2. 确保汇总表格式正确
3. 确保计算文件已保存
4. 在报告中说明共同担保情况
