# 质量检查标准

## 通用质量清单

### 日期验证（所有阶段）

```
□ 从 .processing_config.json 验证破产日期
□ 与前序报告交叉验证（如适用）
□ 在输出中明确记录日期
□ 所有计算使用正确的停止计息日期
```

### 证据标准（事实核查阶段）

```
□ 区分证据材料和申报材料
□ 所有事实引用具体证据来源
□ 时间线按时间顺序排列
□ 不擅自修改申报内容
```

### 计算标准（债权分析阶段）

```
□ 所有计算使用计算器工具
□ 记录计算命令
□ 生成Excel/CSV过程文件
□ 审查LPR期限选择（1年期vs5年期）
□ 验证罚息上限（不超过4倍LPR）
```

### 报告标准（报告整理阶段）

```
□ 准确保留内容（不修改结论）
□ 正确应用模板格式
□ 文件命名符合标准
□ 目录结构完整
```

## 高频错误预防

### 错误1：长期债务LPR期限选择错误

- **问题**：债务期限>5年时使用1年期LPR
- **预防**：必须计算债务期限，考虑使用5年期LPR

### 错误2：日利率换算错误

- **问题**：日利率换算使用错误基数（365 vs 360混淆）
- **预防**：使用直接公式：日利率 × 天数 × 本金

### 错误3：时效中断日期模糊

- **问题**：记录中断为"申报前"而无具体日期
- **预防**：必须记录具体日期并附证据支持
