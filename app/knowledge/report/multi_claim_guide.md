# 多笔债权审查报告编写指南

## 一、适用场景识别

### 1.1 需要分笔处理的情形
当同一债权人存在以下情况时，必须采用分笔债权报告结构：
- 多个独立的合同关系
- 同一债务人的不同法律地位（如：主债务人 + 保证人）
- 多份生效法律文书
- 银团贷款中的份额债权
- 不同担保状态的债权（有财产担保 + 普通债权）

### 1.2 银团贷款特殊处理
银团贷款案件的核心特征：
- 判决原告通常是牵头行/代理行（代表全体银团成员）
- 申报人可能是某一银团成员或其债权受让人
- **必须按份额比例计算可享有债权**

银团份额计算公式：
```
单一成员可享有债权 = 银团总债权 × 该成员份额比例
```

## 三、银团贷款份额债权处理

### 3.1 识别银团贷款案件
在事实核查报告中识别出银团贷款时，必须：
1. 确定银团结构和各成员份额比例
2. 确定申报人受让的是哪个银团成员的债权
3. 按份额比例计算可享有债权金额

### 3.2 份额计算示例

假设：
- 银团总贷款：21.5亿元
- 牵头行A份额：62.79%
- 参加行B份额：13.95%
- 参加行C份额：23.26%
- 判决确认银团总本金：13.02亿元
- 申报人从B行受让债权

则申报人可享有本金 = 13.02亿 × 13.95% = 1.82亿元

### 3.3 报告中的表述

```
根据债权人提供的基础合同材料及[法院名称]作出的已发生法律效力的[案号]民事判决，[牵头行名称]作为银团牵头行和代理行代表全体银团成员获得判决，判决确认[债务人名称]应向[牵头行名称]支付借款本金[银团总本金]元及利息、罚息、复利、违约金等。[申报人的原债权人名称]作为银团成员按份额比例[X]%享有债权本金[份额本金]元。申报人作为[原债权人名称]的债权受让人，通过[债权转让日期]资产买卖协议和[公告日期][报刊名称]公告，合法受让上述债权。
```

## 四、分笔债权本金的正确识别【关键】

### 4.1 本金识别原则

**核心规则：每笔债权的本金必须从对应的债权申报表中单独提取，绝不能简单使用总本金。**

当债权人分多笔申报时，每笔债权都有独立的申报表和金额明细：
- 第一笔债权（有财产担保）的本金 = 从第一份申报表提取
- 第二笔债权（普通债权）的本金 = 从第二份申报表提取
- 两笔本金相加 = 总申报本金

### 4.2 识别信号

看到以下信号时，必须分笔提取本金：
1. 存在多份债权申报表
2. 申报材料分为"有财产担保债权"和"普通债权"两份文件
3. 债权人明确标注"申报两笔债权"
4. 债务人在不同债权中有不同法律地位（主债务人 vs 保证人）

### 4.3 典型案例

**场景**：债权人申报总本金382,374,698.55元，分两笔：
- 第一笔（有担保）：债务人是主债务人，本金265,747,819.63元
- 第二笔（普通）：债务人是保证人，本金116,626,878.92元

**正确处理**：
- 第一笔债权：确认本金265,747,819.63元
- 第二笔债权：确认本金116,626,878.92元

**错误处理**：
- ❌ 第一笔债权：确认本金382,374,698.55元
- ❌ 第二笔债权：确认本金0元

### 4.4 银团贷款份额本金的特别说明

对于银团贷款案件，申报人的本金已经是按份额比例计算后的金额：
- 判决确认银团总本金可能是13亿元
- 但申报人受让的是某一银团成员（如份额13.95%）的债权
- 申报人申报的本金已经是13亿×13.95%≈1.82亿元
- **不要将判决书中的银团总本金与申报人申报的本金混淆**

## 五、常见错误警示

### 5.1 银团贷款常见错误
❌ 将银团总债权作为单一成员的债权确认
❌ 混淆牵头行代理诉讼主体与实际债权持有人
❌ 忽略份额比例计算
❌ 用判决书中的银团总本金替代申报人实际申报的份额本金

### 5.2 多笔债权常见错误
❌ 将多笔债权合并为一笔处理
❌ 漏掉债权结构说明章节
❌ 遗漏汇总确认表
❌ 各笔债权的担保分析混杂
❌ 将总本金全部计入第一笔，第二笔本金写成0

## 六、判决书确认利息的处理【关键】

### 6.1 利息计算结构

判决书通常确认两部分利息：
1. **判决截止日前的利息**：判决书明确载明的具体金额（如"截至2018年5月15日利息为247,674,737.97元"）
2. **判决截止日后的利息**：从判决截止日起至清偿日按约定利率继续计算（如"从2018年5月16日起按合同约定利率计算至实际清偿之日"）

### 6.2 银团贷款利息计算规则

对于银团贷款案件，利息同样需要按份额比例计算：

**计算公式**：
```
单一成员可享有利息 =
  (判决确认的截止日前利息 × 份额比例) +
  (判决截止日后至破产受理日的利息 × 份额比例)
```

**示例**：
- 判决确认银团截止2018.5.15利息：247,674,737.97元
- 申报人份额比例：13.95%
- 申报人份额内截止日前利息：247,674,737.97 × 13.95% ≈ 34,550,406元
- 再加上2018.5.16至破产受理日按约定利率计算的利息

### 6.3 常见错误

❌ 只计算判决截止日后的利息，遗漏截止日前的部分
❌ 使用银团总利息而非份额比例后的利息
❌ 混淆利息、罚息、复利的不同起算规则

## 七、最高额担保限额处理【关键】

### 7.1 最高额保证的概念

当判决确认"保证人在最高额X元范围内承担连带保证责任"时，表示：
- 保证人仅在该限额范围内承担责任
- 超过限额的部分，保证人不承担
- 限额通常包含本金、利息、费用等全部金额

### 7.2 限额计算规则

**当债权总额（本金+利息+费用）超过最高额时**：
- 确认的债权总额应当按最高额封顶
- 各项目金额可按比例分配或优先分配本金

**示例**：
- 判决确认保证人在最高额1.5亿元范围内承担保证责任
- 计算得出本金116,626,878.92元 + 利息36,694,330.89元 = 153,321,209.81元
- 超过最高额1.5亿元（超出3,321,209.81元）
- 应确认的普通债权总额 = 150,000,000.00元（按最高额封顶）

### 7.3 报告表述模板

```
根据判决书确认，江苏熔盛重工有限公司作为保证人在最高额1.5亿元范围内承担保证责任。
本笔债权本金116,626,878.92元、利息及其他费用合计经计算为[X]元，已超过最高额保证限额。
根据最高额限额规定，管理人确认普通债权总额150,000,000.00元（按最高额封顶）。
```
