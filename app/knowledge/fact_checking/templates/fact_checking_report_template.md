# 事实核查报告模板与标准

## 📋 文档说明
本文档为事实核查员专用参考文件，包含：
- 输出格式标准
- 报告模板
- 完整示例

**⚠️ 重要原则：**
- 严格按照债权人申报表内容照抄，不得修改或概括
- 每笔债权独立生成报告，禁止汇总多笔债权
- 完成后移交债权分析员进行深入分析

---

## ⚠️ 模板使用强制规则 (MANDATORY)

**在使用本模板生成报告前,必须阅读并执行以下规则:**

### 占位符替换协议

**规则1: 识别占位符**
- 格式: `[占位符名称]` - 必须替换
- 示例: `[债权人名称]`、`[金额]`、`[日期]`、`[合同编号]`等

**规则2: 定位实际内容**
- 来源: 债权申报表 / 证据材料
- 要求: 内容必须真实存在且可读

**规则3: 替换决策**
```
✅ 内容存在 → 完整照抄实际内容
❌ 内容缺失 → 使用标准缺失标记:
   - [债权人未填写] - 申报表该字段空白
   - [证据未记载] - 证据中无此信息
   - [无法确定] - 证据模糊或矛盾
   - [待补充] - 需要债权人补充材料
```

### ❌ 严禁的占位符填充行为

**禁止1: 常识填充**
```
❌ [金额] → "约100万元" (推测)
❌ [日期] → "2024年左右" (估计)
✅ [金额] → 直接照抄申报表数字,或标记[债权人未填写]
```

**禁止2: 默认值填充**
```
❌ [利率] → "4.35%" (假设为LPR)
❌ [付款方式] → "银行转账" (假设常见方式)
✅ [利率] → 照抄合同条款,或标记[合同未约定]
```

**禁止3: 上下文填充**
```
❌ [签订日期] → "2024年" (因为其他日期都是2024)
❌ [合同编号] → "GY2024-XXX" (仿照其他编号格式)
✅ [签订日期] → 照抄合同载明日期,或标记[合同未载明]
```

**禁止4: 示例文本填充**
```
❌ [债权人名称] → "XX公司" (使用模板示例)
❌ [合同名称] → "《买卖合同》" (使用通用名称)
✅ [债权人名称] → 照抄申报表的准确全称
```

**禁止5: 空白删除**
```
❌ 遇到[缺失项] → 直接删除该行(隐藏缺失)
✅ 遇到缺失项 → 保留该行,填入标准缺失标记
```

### 交付前检查 (MANDATORY)

**步骤1: 搜索所有方括号**
```bash
# 在报告中搜索所有 []
grep -n "\[.*\]" your_report.md
```

**步骤2: 验证每个方括号**
- 每个 `[]` 必须是:
  - ✅ 合法缺失标记 (如 [证据未记载])
  - ✅ 或已替换为实际内容
- 不应是:
  - ❌ 未替换的占位符 (如 [债权人名称])
  - ❌ 推测/假设内容 (如 [约100万])

**步骤3: 确认零推测**
- 报告中所有事实陈述都有证据/申报依据
- 没有使用"应该""可能""大概""根据常理"等推理词汇
- 所有缺失信息都明确标注,未填补

### 占位符使用示例

**示例1: 正常替换**
```
模板: 债权人名称:[债权人名称]
申报表: 江苏XX建筑工程有限公司
报告: 债权人名称:江苏XX建筑工程有限公司 ✅
```

**示例2: 缺失标记**
```
模板: 合同编号:[合同编号]
证据: 合同未载明编号
报告: 合同编号:[合同未载明编号] ✅
```

**示例3: 部分缺失**
```
模板: 付款方式:[付款方式]
证据: 合同约定"分期付款",但未明确期数
报告: 付款方式:分期付款[具体期数:合同未明确] ✅
```

**示例4: 多个缺失**
```
模板: 违约金:[违约金条款]
证据: 合同无违约金条款
报告: 违约金:[合同未约定违约金] ✅
```

### 违反规则的后果

- ❌ 使用推测填充占位符 → 报告包含虚假信息
- ❌ 隐藏缺失信息 → 后续分析员无法识别证据空白
- ❌ 使用示例文本 → 张冠李戴,混淆不同债权人

**请在使用模板前仔细阅读以上规则,在交付报告前执行检查程序。**
