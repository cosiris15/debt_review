# 银团贷款债权审查指南

## 一、银团贷款识别特征

### 1.1 关键识别词
- "银团贷款"/"银团贷款合同"
- "牵头行"/"代理行"/"参加行"
- "份额比例"/"贷款承诺额"
- "银团成员"/"联合贷款"

### 1.2 典型合同结构
- 银团贷款合同：由牵头行代表全体银团成员与借款人签订
- 各银团成员按份额比例承担贷款风险和享有债权

## 二、银团贷款份额计算要点

### 2.1 核心原则
**银团成员仅按其份额比例享有债权，而非全部债权**

### 2.2 份额计算公式
```
单一成员可受让债权 = 判决确认总债权 × 该成员份额比例
```

### 2.3 示例
若银团结构为：
- 牵头行A：份额62.79%
- 参加行B：份额13.95%
- 参加行C：份额23.26%

判决确认总本金为13亿元，则：
- A行可受让债权：13亿 × 62.79% = 8.16亿
- B行可受让债权：13亿 × 13.95% = 1.81亿
- C行可受让债权：13亿 × 23.26% = 3.02亿

### 2.4 债权转让时的份额继承
- 原银团成员转让其份额给AMC时，AMC仅继受该成员的份额债权
- 不能将银团总债权作为受让债权

## 三、银团贷款诉讼特点

### 3.1 代理诉讼机制
- 通常由牵头行/代理行作为原告代表全体银团成员起诉
- 判决中确认的债权是银团总债权，各成员按份额享有

### 3.2 判决主文的理解
判决书中"原告国家开发银行"实际代表全体银团成员，判决确认的债权金额需按份额分配。

## 四、事实核查要点

### 4.1 必须核实的内容
1. 银团成员及其份额比例
2. 债权转让涉及的是哪个银团成员的份额
3. 申报人是否仅申报了其受让的份额债权

### 4.2 常见错误
- 将银团总债权作为单一成员的债权申报
- 混淆牵头行代理诉讼主体与实际债权持有人
- 忽略份额比例计算

### 4.3 审查结论模板
```
本案涉及银团贷款，银团结构如下：
- [银行名称1]：份额XX%
- [银行名称2]：份额XX%
- ...

判决确认银团总债权本金为XXX元，申报人系[银行名称]的债权受让人，按份额比例XX%计算，申报人可享有债权本金XXX元。
```

## 五、与普通债权的区分

### 5.1 单一债权人债权
- 一个债权人，一份债权
- 判决确认多少就是多少

### 5.2 银团贷款债权
- 多个债权人，按份额分享
- 需核实申报人受让的具体份额
- 按份额比例计算可享有债权
